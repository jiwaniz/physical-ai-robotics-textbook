# باب 1: روبوٹک اعصابی نظام (ROS 2)

ROS 2 (Robot Operating System 2) کا استعمال کرتے ہوئے روبوٹ کمیونیکیشن اور کنٹرول کی بنیادیں سیکھیں۔

## باب کا جائزہ

ROS 2 جدید روبوٹکس کی ریڑھ کی ہڈی ہے، جو سینسرز، ایکچویٹرز، اور AI الگورتھمز کو جوڑنے والا کمیونیکیشن انفراسٹرکچر فراہم کرتا ہے۔ اسے روبوٹ کے "اعصابی نظام" کے طور پر سمجھیں۔

### آپ کیا سیکھیں گے

**ہفتہ 3: ROS 2 آرکیٹیکچر اور بنیادیں**
- ROS 2 کے تصورات: nodes، topics، services، actions
- Ubuntu 22.04 پر ROS 2 Humble سیٹ اپ کرنا
- اپنا پہلا ROS 2 node بنانا
- Publish-subscribe پیٹرن کو سمجھنا

**ہفتہ 4: Nodes، Topics، اور کمیونیکیشن**
- Topics پر publishing اور subscribing
- Custom message types
- Service client/server patterns
- Quality of Service (QoS) settings

**ہفتہ 5: Packages اور Launch Files**
- ROS 2 packages بنانا
- Launch file configuration
- Parameter management
- ROS 2 applications کو build اور run کرنا

## سیکھنے کے نتائج

اس باب کے اختتام تک، آپ یہ کر سکیں گے:
- ✅ ROS 2 Humble کو install اور configure کرنا
- ✅ Python میں ROS 2 nodes بنانا
- ✅ Publish-subscribe communication کو implement کرنا
- ✅ Custom ROS 2 packages بنانا
- ✅ پیچیدہ روبوٹ سسٹمز کے لیے launch files استعمال کرنا

## عملی پراجیکٹس

- ایک سادہ publisher-subscriber سسٹم بنانا
- روبوٹ کنٹرول کے لیے ایک service بنانا
- مکمل ROS 2 application کو package کرنا

## تشخیص

**پراجیکٹ: ROS 2 Package Development**
- ایک multi-node ROS 2 سسٹم ڈیزائن اور implement کریں
- Nodes کے درمیان کمیونیکیشن کا مظاہرہ کریں
- اپنے package کو README اور launch files کے ساتھ دستاویز کریں
